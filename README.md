## Prerequisites
- Java

## Installation
1. Clone the repo
```sh
git clone https://github.com/Senull0/ChatB.git
```
2. Install dependencies
3. Set your API key in Chatdata
```sh
KEY=" "
```
4. Run the app (API version 32 or higher recommended)
